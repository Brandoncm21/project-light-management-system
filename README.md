# Project Light - High-End Cinema Equipment Management System

## Project Overview
This is a comprehensive management platform developed for **Project Light Films**, a company specializing in high-end cinematography equipment rentals. The system streamlines the entire rental lifecycle, ensuring professional handling of expensive assets and client reservations.

## Key Features & Contributions
* **Full-Stack Development:** Developed robust backend logic using .NET for inventory tracking and booking management.
* **Cross-Platform Mobile App:** Integrated a mobile solution built with **FlutterFlow** and **Firebase** to allow real-time equipment availability checks.
* **Advanced Security:** Implemented secure user authentication and credential protection for administrative and client access.
* **DevOps & Quality Assurance:** Managed version control via GitHub and performed rigorous QA testing, including bug tracking and security audits.
* **Cloud Deployment:** Orchestrated the application deployment using **Microsoft Azure** services.

## Technical Stack
* **Backend:** .NET Core / C#
* **Mobile:** FlutterFlow & Firebase
* **Database:** SQL Server (Relational Design)
* **Cloud:** Microsoft Azure
* **VCS:** Git & GitHub
